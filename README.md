# Welcome to the Magna API

A backend API for Creative Lab's Liberty In North Korea Project

Check out **Creative Labs** at:
	uclacreatives.com

A short guide to run:


```
	**clone repo**
	$ cd magna
	$ python3 -m venv env
	$ source env/bin/activate
	$ cd magna
	$ make install
	$ make migrate
	$ make test
	$ make dev
```
